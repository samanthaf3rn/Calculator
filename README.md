# Calculator
A simple menu driven calculator program using Python

Features
Addition: Sum two numbers.
Subtraction: Find the difference between two numbers.
Multiplication: Multiply two numbers.
Division: Divide numbers with built-in zero-division error handling.
Persistent Loop: The menu stays open until the user chooses to exit.

Requirements
Python 3.6 or higher installed on your system.

How to Use
1. Clone or Download the script file calculator.py
2. Open your terminal or command prompt.
3. Navigate to the folder containing the script.
4. Run the program using the following command 
  python calculator.py
5. Follow the on-screen prompts to select an operation and enter your numbers.
